# 数据库管理系统（简化版）

一个基于 Vue 3 + Vite 的数据库管理系统，基于 Vue 3 的 Composition API 构建。 的数据库管理前端系统。

## 功能特性

- 📊 **数据表管理**：查看数据库中的所有表、表结构
- 🔍 **SQL查询**：执行SQL查询，查看结果
- 📈 **图表分析**：简单的数据可视化图表
-    **导入/导出数据库** 导入/导出数据库
- 🎨 **历史记录**：操作记录、sql记录

## 技术栈

- Vue 3 (Composition API)
- Vue Router 4
- Element Plus
- Axios
- Vite
